## 🌱 開始動手

本課程包含建構 AI 代理基礎的課程。每課涵蓋不同主題，歡迎隨意開始！

此課程提供多語言支援，請參考[可用語言列表](#-多語言支援)。

若您是首次使用生成式 AI 模型構建，請參考我們的 [初學者生成式 AI 課程](https://aka.ms/genai-beginners)，內含 21 節建構 GenAI 的課程。

別忘了 [為本儲存庫加星（🌟）](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) 並 [分支本儲存庫](https://github.com/microsoft/ai-agents-for-beginners/fork) 以執行程式碼。

### 認識其他學習者，獲得問題解答

如有任何關於建構 AI 代理的問題或卡住，歡迎加入我們專門的 Discord 頻道，位於 [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord)。

### 您需要準備的

本課程每課均包含程式碼範例，位於 code_samples 資料夾。您可 [分支本儲存庫](https://github.com/microsoft/ai-agents-for-beginners/fork) 創建自己的副本。  

這些練習程式碼範例使用 Microsoft Agent Framework 與 Azure AI Foundry Agent Service V2：

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) — 需要 Azure 帳戶

本課程使用來自 Microsoft 的以下 AI 代理框架與服務：

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


欲瞭解如何執行本課程程式碼，請參考 [課程設定](./00-course-setup/README.md)。
