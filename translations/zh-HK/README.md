## 🌱 入門

本課程包含建構 AI 代理人的基本課程。每個課程涵蓋不同主題，可以從任意課程開始學習！

本課程支援多語言。請參考我們的[支援語言列表](#-multi-language-support)。

如果你是第一次要用生成式 AI 模型來建構，請查看我們的 [生成式 AI 初學者課程](https://aka.ms/genai-beginners)，包含 21 課教你如何使用 GenAI。

別忘了[給本存儲庫點星 (🌟)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst)和[將本存儲庫分支 (fork)](https://github.com/microsoft/ai-agents-for-beginners/fork)以執行程式碼。

### 遇見其他學習者，解答你的問題

若你遇到困難或有 AI 代理人建構相關問題，歡迎加入我們專門的 Discord 頻道，在 [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) 。

### 你需要準備什麼

本課程每堂課都包含程式範例，程式碼範例位於 code_samples 資料夾。你可以[將本存儲庫分支(fork)](https://github.com/microsoft/ai-agents-for-beginners/fork)創建自己的副本。

這些練習中使用的程式範例基於 Microsoft Agent Framework 和 Azure AI Foundry Agent Service V2：

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - 需要 Azure 帳戶

本課程使用 Microsoft 提供的以下 AI 代理人框架與服務：

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


若要了解更多關於如何執行本課程程式碼，請參閱 [課程設置](./00-course-setup/README.md)。
